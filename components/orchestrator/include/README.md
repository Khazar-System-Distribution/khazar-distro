# include — Ümumi tiplər və təriflər

- Bütün modullar tərəfindən istifadə olunan ümumi məlumat strukturlarını, sabitləri və makroları ehtiva edir.
- Tip təyinatları (`typedef`), struktur tərifləri və enum sabitləri burada mərkəzləşdirilir.
- Başlıq faylları vasitəsilə layihə daxilində tip uyğunluğunu təmin edir.
